[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Let's look under the hood at how these agents collaborate.
(point or gesture)
First, the user inputs their raw utility data. But before any AI processing happens, the data passes through our custom Security Checkpoint, which instantly scrubs any sensitive PII like SSNs or Aadhaar numbers and blocks malicious prompt injections.
Once cleared, our Ask Country Node pauses the flow to explicitly ask the user for their location, ensuring our calculations are perfectly localized.
Then, the Data Analyst Agent takes over to extract and summarize the usage numbers. 
This agent passes the baton to the Climate Impact Agent, which calculates the exact carbon footprint. 
Both of these agents are supercharged by a dedicated MCP Server that fetches real-time, deterministic emission factors and baseline data.
Finally, everything stops at the Human Approval Pause Node, giving the user final sign-off before the official Eco Report is generated.
